# Bistri EngineeringThis is a repo for managing machine readable artifacts for the [Bistri Engineering APIs](http://developers.bistri.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Bistri Engineering APIs](http://developers.bistri.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://developers.bistri.com)- [Developer](http://developers.bistri.com)- [Twitter](https://twitter.com/BistriEng)The [APIs.json](https://github.com/api-evangelist/bistri-engineering/blob/master/apis.json) index provides links to all the relevant links for the Bistri Engineering APIs, including the OpenAPI and Postman if they are present.
